Learned Sparse Retrieval Framework

A framework for training and evaluating various LSR architectures.

Binary Encoder

The binary encoder assigns a weight of 1 to terms in the input query or document and 0 to the rest.

MLP Encoder

The MLP encoder uses a Linear layer on top of the Transformer's last hidden states to score terms in the input sequence.

expMLP Encoder

ExpMLP requires a pre-processing step to expand documents with relevant terms, and then uses the MLP encoder to store both original and expanded terms

MLM Encoder

The MLM encoder uses the MLM head in the Transformer Masked Language Model to score terms and expand the input to relevant terms, end-to-end.

CLS_MLM Encoder

CLS_MLM encoder uses the same architecture as the MLM encoder; however, only the logit of the CLS token is used for term weighting and expansion.
